# calculadora-python

Uma calculadora feita em Python com avaliação segura de expressões usando `ast`.  
Permite operações matemáticas básicas, funções trigonométricas, raízes e muito mais.

---

##  Funcionalidades

- Operações: `+`, `-`, `*`, `/`, `//`, `%`, `**`
- Suporte a números inteiros e decimais
- Funções matemáticas:  
  `sin`, `cos`, `tan`, `sqrt`, `log`, `log10`, `exp`, `abs`, `round`
- Modo interativo (REPL)
- Avaliação segura (não permite código malicioso)
- Testes unitários opcionais

---

###  1. Rodar no modo interativo
